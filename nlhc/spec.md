# 自然言語にプログラムの考えを

意思疎通のための言語にプログラミングの考え方をぶち込む

オブジェクト指向 型 型変換 型推論

## 文
```
class 文 {
}
```

## ノード
```
class verbnode {
    args: 引数, // 目的語
    tense: 時制, // 過去 現在 未来
    state: 状態, // 継続 断続 起動 完了 結果 終止
    view: 態, // 受動 能動 交互
    sence: 感覚, // 使役 自発 尊敬
    meaning: 意味,
}
class nounnode {
    meaning: 意味,
    case: 格, // 主格 所有格 与格 奪格
    args: 引数, // 目的語
}
class adjectivenode {
    meaning: 意味,
    args: 引数, // 名詞句
}
class conjunctionnode {
    meaning: 意味,
    args: 引数, // 名詞句
}
```

## 単語
単語の意味は、階層構造で表される
```
class word {
    name: 名前,
    meaning: {
        verb: {
            "意味": {
                意味,
                意味,
            },
            "意味": {
                意味,
                意味,
            },
        },
        noun: {
            "意味",
        },
    }
}
```
